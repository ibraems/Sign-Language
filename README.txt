For this Image Classification task use the sign language dataset. The data is related to sign languages (also known as signed languages) are languages that use manual communication to convey meaning. This can include simultaneously employing hand gestures, movement, orientation of the fingers, arms or body, and facial expressions to convey a speaker's ideas. 

Details of datasets:
Image size: 64x64
Color space: Grayscale
File format: npy
Number of classes: 10 (Digits: 0-9)
Number of participant students: 218
Number of samples per student: 10

The classification goal is to predict the what digit each sign image represents.
